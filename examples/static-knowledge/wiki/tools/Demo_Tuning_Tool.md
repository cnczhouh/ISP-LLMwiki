# Demo_Tuning_Tool

Demo_Tuning_Tool 是一个脱敏工具页示例，用于说明工具入口、观察指标和相关页面如何组织。

## 页面属性
- 类型：通用工具
- 厂家：Demo Vendor
- 平台：[[wiki/platforms/DemoSensor|DemoSensor]]
- 场景：参数查看、亮度调试、问题复现记录
- 适用范围：示例平台

## 使用场景
- 查看 AE target、曝光时间和 gain。
- 保存问题场景截图和参数快照。
- 配合 [[wiki/workflows/图像问题排查流程|图像问题排查流程]] 记录调试过程。
