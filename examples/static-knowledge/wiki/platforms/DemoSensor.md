# DemoSensor

DemoSensor 是一个脱敏示例平台，用于说明图像调试知识库如何组织平台、模块、问题和流程。

## 平台属性
- 类型：平台
- 主类型：Sensor
- 附加属性：集成 ISP / HDR / 低照优化
- 厂家：Demo Vendor
- 平台：[[wiki/platforms/DemoSensor|DemoSensor]]
- 场景：静态部署示例、平台资料导航
- 适用范围：示例平台

## 平台定位
- 用于演示 GitHub Pages 静态站点的知识页面生成。
- 只保留公开示例，不包含真实厂商资料或客户项目记录。
- 页面之间通过双向链接形成最小可浏览网络。
