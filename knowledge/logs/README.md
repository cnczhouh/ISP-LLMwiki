# 知识库维护日志说明

`logs/` 用来记录知识库如何演化，重点记录资料摄取、页面重构、索引调整和 Lint 审查结果。

## 使用原则
- 日志只记录过程，不替代 `raw/` 原始来源，也不替代 `wiki/` 稳定结论
- 批量处理资料、创建多个页面、重构索引或执行 Lint 审查后，追加日志
- 小范围修 typo、改链接、补一句说明，可以不写日志
- 无法判断的问题写入“待确认”，不要强行改成确定结论

## 推荐文件
- `logs/YYYY-MM.md`：按月记录摄取和审查
- `schema/ingest_log_template.md`：摄取日志模板
- `schema/lint_review_template.md`：健康检查 / Lint 审查模板
