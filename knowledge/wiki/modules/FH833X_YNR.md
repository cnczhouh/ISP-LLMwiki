# FH833X_YNR

FH833X YNR 亮度去噪模块，用于补偿 NR3D / NR2D 后残留的亮度噪声，并在锐化之前抑制噪声被 APC 放大。

## 页面属性
- 类型：平台模块
- 厂家：Fullhan / 富瀚微
- 平台：[[wiki/platforms/FH833X|FH833X]]
- 模块：YNR / 亮度降噪 / coeff 联动
- 场景：亮度噪声、运动区域降噪、噪声回加、锐化前降噪、车载弱 3DNR
- 适用范围：指定平台

## 作用
- 对高 / 中 / 低频亮度噪声进行分频段处理。
- 结合 NR3D coeff 对运动区域做联动降噪。
- 通过 Merge / 噪声回加控制静止区细节和运动区噪声形态。

## 关键参数
- `ynr_en`：YNR 策略开关。
- `ynr_mode`：manual / gainMapping。
- `ctrl_mode_0/1/2`：高 / 中 / 低频去噪强弱模式。
- `ynr_th_slope0/1/2`：高 / 中 / 低频映射斜率，`0x10` 为 1 倍；大于 1 会变锐，小于 1 会变糊。
- `ynr_th_str0/1/2`：高 / 中 / 低频去噪等级。
- `thl0_0~15`、`thl1_0~15`、`thl2_0~15`：按 16 档亮度配置各频段去噪阈值。
- `coef_en`：YNR 与 NR3D coeff 联动使能。
- `mt_coeff0`、`mt_coeff1`：静止 / 小运动 / 大运动分界点。
- `ynr_mt_str0`、`ynr_mt_str1`：小运动 / 大运动联动去噪强度。
- `ynrMergeEn`：噪声回加 / 融合使能。
- `ynrMergeThl/Thh`、`ynrMergeThlY/ThhY`：按 coeff 控制融合阈值和权重。

## 调试视角
YNR 是 NR3D / NR2D 后、APC 前的亮度噪声整形模块，实际效果取决于频段阈值、亮度分段、运动 coeff 联动和噪声回加一起作用。

- 高频 YNR 主要压细颗粒噪声，过强会让纹理变糊；中频 YNR 会影响块状噪声和皮肤质感，过强容易蜡像感；低频 YNR 一般慎用，过强会让大面积明暗过渡不自然。
- `ynr_th_slope` 不只是降噪强度，偏大还会改变锐利感；尤其 `ynr_th_slope1` 增大时容易把中频边界变成毛刺。
- 16 档亮度阈值决定不同亮度区域的噪声形态。暗部调强能压黑场噪声，但会损暗部纹理；亮部调强可能让天空、墙面更干净，但细节层次变少。
- `mt_coeff0/1` 与 NR3D coeff 配合决定静止、小运动、大运动的分界。分界太低会把噪声当运动加强降噪；分界太高会让运动区域残留颗粒。
- `ynrMergeEn` 是细节和噪声形态的回加工具。回加多则细节和颗粒都回来，回加少则干净但塑料感明显。

### 典型调试取舍
| 目标 | 常见调整方向 | 图像收益 | 主要副作用 |
| --- | --- | --- | --- |
| 暗部更干净 | 增强暗部高频 / 中频阈值 | 黑场颗粒下降 | 暗部纹理、毛发、布料细节损失 |
| 保留静止细节 | 开启 merge 并增加静止区回加 | 纹理自然，APC 更容易锐化 | 细噪回加，画面不够干净 |
| 压运动噪声 | 用 coeff 联动提高运动区 YNR | 运动区域颗粒减少 | 运动物体发糊、边缘被抹 |
| 车载弱 3DNR | 弱化或关闭 coeff 差异 | 行进画面一致性更好 | 静止 / 运动区不能分别优化 |

## 基础调试步骤
1. 打开 `ynr_en`，通常 `ctrl_mode` 可配置为强模式以消除亮度噪声。
2. `ynr_th_slope` 保持 `0x10`，先关闭 `coef_en` 和 `ynrMergeEn` 调基础去噪。
3. 通过各亮度档阈值调同一场景不同亮暗区域的高中低频去噪效果。
4. 在不同照度下调 gainMapping，低照下主要去高频噪声，可去部分中频噪声，低频通常设为 0。
5. 低照可用 `ynr_th_slope2` 轻微抬升锐化；`ynr_th_slope0/1` 尽量保持 `0x10`，尤其 `ynr_th_slope1` 增大易引起边界毛刺。

## 联动和回加
1. 打开 `coef_en`，用 `mt_coeff0/1` 区分静止、小运动、大运动；常见范围 `mt_coeff0=0x38~0x60`、`mt_coeff1=0xb0~0xe0`。
2. 调 `ynr_mt_str0` 控制小运动 / 噪声区域去噪，调 `ynr_mt_str1` 控制大运动物体去噪。
3. 打开 `ynrMergeEn` 后，可按 NR3D coeff 控制静止 / 运动区域噪声回加，保留静止区细节。
4. 低照时，噪声和 coeff 逐步增大，应同步增大 merge 阈值，并适当减小权重以控制噪声形态。

## 车载弱 3DNR 注意
- 车载项目可能采用非常弱的 NR3D，甚至关闭 NR3D。
- 此时可关闭 `coef_en/ynrMergeEn`，或将全部 coeff 等级对应权重设为相同，不区分运动和静止。
