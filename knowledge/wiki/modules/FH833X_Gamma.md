# FH833X_Gamma

FH833X Gamma 模块用于对图像做非线性映射，包含 YGamma 和 CGamma 两类曲线，影响亮度、色彩、对比度、通透性、噪声与清晰度观感。

## 页面属性
- 类型：平台模块
- 厂家：Fullhan / 富瀚微
- 平台：[[wiki/platforms/FH833X|FH833X]]
- 模块：Gamma / YGamma / CGamma
- 场景：亮度映射、对比度、通透性、昼夜曲线插值、WDR 动态范围配合
- 适用范围：指定平台

## 作用
- YGamma：在亮度域做 160 阶映射。
- CGamma：可在 RGB 域或 YC 域作用；RGB 域会同时影响亮度和色度，YC 域 C 通道只影响色度。
- 手册建议一般使用 RGB 域 CGamma，即 `gamma_mode=2`，不使用 YGamma。

## 关键参数
- `gamma_en`：Gamma 策略总开关。
- `gamma_mode`：Gamma 模块输入选择。
  - `0`：关闭 Gamma。
  - `1`：开启 YGamma 和 CGamma，但 CGamma 只影响色度分量。
  - `2`：开启 RGB 域 CGamma，关闭 YGamma，默认推荐。
  - `3`：同时开启 YGamma 和 RGB 域 CGamma。
- `y_ctrl_mode`：YGamma 表模式，`3` 为用户自定义表。
- `c_ctrl_mode`：CGamma 表模式，`3` 为用户自定义表。
- `cGammaMode`：两张 CGamma 表选择 / 内插模式。
- `gammaTableWeight`：两张 CGamma 表内插权重。
- `refresh`：模式或表配置刷新位。

## 调试步骤
1. 先选择 `gamma_mode`，通常使用 `gamma_mode=2`。
2. 调试时可将 y/c 控制模式设为自定义表，通过 CoolView 工具实时拉曲线。
3. 用 Gamma 让整体亮度和对比度达到预期，但不要替代 AE / DRC 的主职责。
4. 当一条曲线不能同时兼顾白天和夜晚时，可开启 CGamma 内插：白天一条曲线、暗态一条曲线，中间 db 通过权重融合。

## 调试权衡
Gamma 曲线调试的核心是分亮度段决定“抬亮、压暗、增强对比或保留层次”，它会同时改变噪声、色彩和锐化观感。

- 暗部曲线拉高会让暗部细节更容易看见，但黑位被抬起后会发灰，NR 残留噪声也更明显。
- 中间调斜率增大能提升主体对比和通透性，但肤色、墙面、车身等大面积区域可能变硬，亮度过渡不柔和。
- 高亮段压缩能保住天空、灯牌和车灯层次，但压得太多会让高光发灰、不通透。
- RGB 域 CGamma 会同时影响亮度和色度，曲线变化可能改变饱和度观感；如果颜色异常，要联查 CCM、CE 和 DRC 饱和度。
- 白天 / 夜晚两条曲线内插时，权重变化必须平滑，否则会出现亮度、对比度或色彩风格随 gain 跳变。
- WDR 场景先用 DRC 解决动态范围主矛盾，再用 Gamma 做整体观感微调，避免用 Gamma 强行拉暗部导致噪声和发灰。

### 典型调试取舍
| 目标 | 常见调整方向 | 图像收益 | 主要副作用 |
| --- | --- | --- | --- |
| 暗部更亮 | 拉高暗部曲线 | 暗部细节可见 | 发灰、噪声上升 |
| 画面更通透 | 增大中间调斜率、稳住黑位 | 主体对比增强 | 过渡变硬，暗部可能丢层次 |
| 高光更柔和 | 压缩高亮段 | 高光层次保留 | 高光发灰、亮感下降 |
| 昼夜风格连续 | 使用双曲线平滑内插 | 切换自然 | 曲线差异过大会跳变 |
