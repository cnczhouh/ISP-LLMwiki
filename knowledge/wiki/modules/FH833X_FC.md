# FH833X_FC

FH833X FC（False Color）去伪彩模块，用于去除 CFA 内插过程引入的伪彩色，尤其关注高频纹理区域的假色。

## 页面属性
- 类型：平台模块
- 厂家：Fullhan / 富瀚微
- 平台：[[wiki/platforms/FH833X|FH833X]]
- 模块：FC / False Color / 去伪彩
- 场景：CFA 插值伪彩、高频纹理假色、颜色误伤权衡
- 适用范围：指定平台

## 作用
- 去除 CFA 内插过程引入的伪彩色。
- 主要用于高频纹理场景，例如细密线条、织物、栅格等位置出现的异常彩色边或彩色纹理。

## 关键参数
- `fc_en`：FC 策略总开关，在 `FUNC_CTRL` 中。
- `offset_t`：去假色纹理信息偏移量，值越大去假色强度越弱；值越大，去假色越偏向作用于更高频纹理区域；为 0 时作用于所有纹理区域，默认 `0xfff`。
- `offset_y`：去假色亮度信息偏移量，值越大去假色强度越弱；值越大，去假色越偏向作用于更高亮纹理区域；为 0 时作用于所有亮度区域，默认 `0xfff`。
- `str`：去假色强度，值越大去假色越强，但更容易导致正常颜色变灰。

## 调试步骤
1. 搭建或选择存在高频分量的场景，观察高频假色是否明显。
2. 调整 `offset_t` 和 `offset_y`，确认 FC 作用区域是否覆盖伪彩区域。
3. 同时观察场景中其他正常颜色物体是否被误伤。
4. 调 `str` 控制去假色强度，直到伪彩和颜色保真之间达到平衡。

## 调试权衡
FC 调试需要先区分“CFA 伪彩”和“真实彩色纹理”，再决定作用区域和去色强度。

- `offset_t` 越小，FC 覆盖的纹理范围越大，假色更容易被压掉，但正常细彩色纹理也更容易变灰；越大则只处理更高频区域，误伤少但假色可能残留。
- `offset_y` 越小，更多亮度区域参与去假色，暗部和中灰纹理也会被处理；越大则偏向高亮纹理，暗部假色可能漏掉。
- `str` 越强，彩色边和高频伪彩越少，但彩色细节、织物颜色和边缘饱和度会下降。
- APC 增强高频后会让伪彩更显眼，CE / CCM 提饱和后也会放大假色观感，所以 FC 不能脱离锐化和色彩模块单独判断。

### 典型调试取舍
| 目标 | 常见调整方向 | 图像收益 | 主要副作用 |
| --- | --- | --- | --- |
| 高频假色更少 | 降低 `offset_t/offset_y`，提高 `str` | 伪彩边减少 | 正常彩色纹理变灰 |
| 保留彩色细节 | 提高 offset 或降低 `str` | 色彩细节更自然 | 栅格、织物假色残留 |
| 控制锐化后假色 | 先定 APC，再小幅加强 FC | 边缘彩噪减少 | 清晰度和彩色边缘饱和度下降 |
