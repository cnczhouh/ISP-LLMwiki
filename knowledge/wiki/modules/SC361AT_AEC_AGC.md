# SC361AT_AEC_AGC

SC361AT AEC / AGC 功能整理，用于记录 datasheet 中自动曝光、自动增益、HDR 分支曝光 / 增益寄存器和约束关系。

## 页面属性
- 类型：平台模块
- 厂家：SmartSens / 思特威
- 平台：[[wiki/platforms/SC361AT|SC361AT]]
- 模块：AEC / AGC
- 场景：自动曝光、自动增益、HDR 分支曝光比例、亮度目标、低照增益控制
- 适用范围：指定平台

## 模块作用
- AEC / AGC 都基于亮度调节。
- AEC 调节曝光时间，AGC 调节增益值，最终使图像亮度落在设定亮度阈值范围内。
- SC361AT 的 AEC / AGC 策略是优先打开曝光时间，当曝光时间达到最大值时再开始调整增益，直到增益达到最大值。
- HDR 模式下需要同时考虑 LE、SE、HCG、LCG、OF 等分支的曝光与 gain 约束。

## AEC 曝光控制
| 功能 | 寄存器 | 说明 | 调节步长 | 最小值 | 最大值 |
|---|---|---|---:|---:|---:|
| 手动模式 | `16'h5e02[3:0]` | `4'h0` 自动曝光，`4'hf` 手动曝光 | - | - | - |
| LE 曝光时间 | `{16'h5e36, 16'h5e37}` | 长曝光手动曝光时间，寄存器值以 1 行为单位 | 4 | 257 | `{16'h320e,16'h320f} - {16'h5e4e,16'h5e4f} - 0x40` |
| SE 曝光时间 | `{16'h5e4e, 16'h5e4f}` | SE 时间，寄存器值以 1 行为单位 | 1 | 1 | `16'h88` |

## AGC 增益控制
| 功能 | 寄存器 | 最小值 | 最大值 | 说明 |
|---|---|---:|---:|---|
| 手动模式 | `16'h5e02[3:0]` | - | - | `4'h0` 自动增益，`4'hf` 手动增益 |
| HCG 帧增益 | `{16'h5e32, 16'h5e33}` | `16'h200` | `16'hffff` | HCG 帧增益，单位 1/64x |
| LCG 帧增益 | `{16'h5e3a, 16'h5e3b}` | `16'h40` | `16'h4000` | LCG 帧增益，单位 1/64x |
| OF 帧增益 | `{16'h5e42, 16'h5e43}` | `16'h56` | `16'h4000` | OF 帧增益，单位 1/64x |
| SE 帧增益 | `{16'h5e4a, 16'h5e4b}` | `16'h40` | `16'h4000` | SE 帧增益，单位 1/64x |

## AEC / AGC 约束关系
- `LCG增益 * 8 ≤ HCG增益 ≤ LCG增益 * 64`
- `OF增益 ≤ LCG增益 * OF ratio ≤ OF增益 * 64`
- `SE曝光 * SE增益 ≤ LE曝光 * OF增益 / OF ratio ≤ SE曝光 * SE增益 * 64`
- `LE曝光 * HCG增益 / SE曝光 / SE增益 ≤ 4096`
- `OF ratio` 的值根据量产情况决定。

## 调试建议
- 先确认当前是自动模式还是手动模式，避免以为 AEC / AGC 生效但实际由手动寄存器锁定。
- HDR 亮度异常时，不要只看 LE 曝光，还要同时检查 HCG、LCG、OF、SE 分支 gain 和约束关系。
- 低照噪声大时，先判断是否已经达到最大曝光后转入高 gain，再联查 [[wiki/modules/SC361AT_HDR|SC361AT_HDR]] 和后级 PWL / LTM。
- 高光过曝时，检查 SE 曝光、OF gain 和 HDR 合成余量，而不是只降低整体曝光。
