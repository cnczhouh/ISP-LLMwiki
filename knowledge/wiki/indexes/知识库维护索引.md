# 知识库维护索引

## 页面属性
- 类型：索引
- 厂家：通用
- 平台：跨平台
- 模块：知识库维护 / 摄取日志 / Lint 审查
- 场景：资料摄取、知识库审查、索引维护
- 适用范围：跨平台

## 用途
- 记录资料从 `raw/` 进入 `wiki/` 的整理过程
- 记录批量更新、索引调整和结构重构
- 定期检查重复页面、孤立页面、缺失来源、断链和旧结论冲突
- 把暂时无法判断的问题留在“待确认”，避免误写成稳定结论

## 维护入口
- [[logs/README|维护日志说明]]
- [[logs/2026-05|当前维护日志]]
- [[logs/2026-04|历史维护日志]]
- `schema/ingest_log_template.md`：摄取日志模板（内部模板，不在 Web 展示）
- `schema/lint_review_template.md`：Lint 审查模板（内部模板，不在 Web 展示）

## 什么时候写日志
- 批量处理 raw 资料
- 新建多个 wiki 页面
- 重构平台、模块、问题或索引结构
- 合并、重命名、删除页面
- 执行一次知识库 Lint 审查

小范围修 typo、补一个链接、改一句描述，可以不强制写日志。

## Lint 审查重点
- 重复或同义页面
- 孤立页面和缺少索引入口的页面
- 页面属性缺失或不一致
- 重要结论缺少 raw 来源或案例来源
- 模块页缺少调试视角、参数联动、典型调试取舍
- 问题页缺少现象、原因、排查顺序、判断依据、误判
- 平台特有内容和通用知识混写
- 索引遗漏新页面或链接到旧页面
- 新资料修正或推翻旧结论
