# FH833X_图像质量调试流程

FH833X 图像质量调试流程，用于把 sensor 接入、基础标定、线性模式调优、WDR 模式调优和常见问题排查串成可执行顺序。

- 场景：FH833X bring-up、线性模式图像质量调试、WDR 图像质量调试、问题定位
- 适用范围：指定平台

## 调试前置条件
- 已拿到 sensor datasheet、配置文件、曝光配置表和必要接口说明。
- sensor driver 已接入 FULLHAN 平台，基本出图正常。
- 帧率、分辨率、输出格式符合项目需求。
- AE 能控制 sensor 曝光时间和 gain，且生效时序可确认。
- 亮态和暗态均能正常出图。

## 总体顺序
1. Sensor 接入与基础通路确认。
2. Sensor 与镜头基础标定：[[wiki/modules/FH833X_BLC|FH833X_BLC]]、[[wiki/modules/FH833X_LSC|FH833X_LSC]]、AWB 白点、[[wiki/modules/FH833X_CCM|FH833X_CCM]]。
3. 亮度调优：[[wiki/modules/FH833X_AE|FH833X_AE]] 为主，[[wiki/modules/FH833X_Gamma|FH833X_Gamma]] / [[wiki/modules/FH833X_DRC|FH833X_DRC]] 为辅。
4. 色彩调优：[[wiki/modules/FH833X_AWB|FH833X_AWB]] -> [[wiki/modules/FH833X_CCM|FH833X_CCM]] -> [[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]] / [[wiki/modules/FH833X_CNR|FH833X_CNR]] / [[wiki/modules/FH833X_PURPLE|FH833X_PURPLE]] / [[wiki/modules/FH833X_FC|FH833X_FC]] / [[wiki/modules/FH833X_HLR|FH833X_HLR]]。
5. 对比度和通透性调优：[[wiki/modules/FH833X_Gamma|FH833X_Gamma]]、DRC、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、[[wiki/modules/FH833X_YIE|FH833X_YIE]]。
6. 清晰度和噪声调优：[[wiki/modules/FH833X_NR|FH833X_NR]]、[[wiki/modules/FH833X_APC|FH833X_APC]]、[[wiki/modules/FH833X_YNR|FH833X_YNR]]、[[wiki/modules/FH833X_CNR|FH833X_CNR]]、[[wiki/modules/FH833X_DPC|FH833X_DPC]]，并参考 [[wiki/modules/FH833X_SHARP_NOISE|FH833X_SHARP_NOISE]]。
7. WDR 模式专项调优：[[wiki/modules/FH833X_WDR|FH833X_WDR]]、AE、DRC、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、NR、APC 联调。
8. 常见问题回归：闪烁、偏色、噪声、拖影、锐化、HDR 伪影。

## 1. Sensor 接入
### 完成标准
- 基本通路正常。
- 帧率和分辨率符合需求。
- AE 可以控制 sensor 曝光时间和 gain。
- 曝光 / gain 时序正常，尽量同帧生效，避免闪烁。
- sensor 驱动默认参数合理。
- 亮暗态均可正常出图。

### 注意事项
- 最大 / 最小曝光时间和 gain 必须符合 sensor datasheet。
- sensor gain 若不连续，可通过 sensor dgain 补偿，或用 `CalcSnsValidGain()` 将真实 gain 反馈给 ISP。
- sensor 曝光时间若不连续，可用 `CalcSnsValidIntt()` 将真实曝光时间反馈给 ISP。

## 2. Sensor 与镜头标定
### BLC
- 参考 [[wiki/modules/FH833X_BLC|FH833X_BLC]]。
- BLC 是 ISP 调试第一步，黑电平不准会影响后续颜色、对比度和噪声判断。
- BLC 值会跟随 sensor gain 漂移，建议做 gain 联动。

### LSC
- 参考 [[wiki/modules/FH833X_LSC|FH833X_LSC]]。
- LSC 用于补偿镜头暗角和 color shading。
- 按 CoolViewPro2 LSC 标定流程生成结果，并通过 `lsc_table.c` 配置给芯片。

### AWB 白点
- AWB 白点与 sensor、镜头和滤光片强相关。
- 更换 sensor 或镜头滤光片后必须重新标定。
- 参考 [[wiki/modules/FH833X_AWB|FH833X_AWB]] 做白点、白框、色温框和场景权重配置。

### CCM
- 参考 [[wiki/modules/FH833X_CCM|FH833X_CCM]]。
- CCM 用于将 sensor RGB 响应校正到接近人眼观感。
- 采集 24 色卡时要保证灰阶不过曝，DRC 饱和度抑制先不要误伤颜色。
- CCM 矩阵每行和应为 1，避免影响白平衡。

## 3. 亮度调优
- 在固定色温光源下，先用手动 WB 和 CCM 保证基础颜色正确。
- 预设 [[wiki/modules/FH833X_Gamma|FH833X_Gamma]] 和 DRC 参数，主要调 [[wiki/modules/FH833X_AE|FH833X_AE]]。
- AE 调试顺序：参考亮度 -> 统计模式 -> 稳定区间 -> 自动曝光开关 -> 高级模式。
- 打开精细曝光后，必须确认 intt、again、dgain 生效时序。
- [[wiki/modules/FH833X_Gamma|FH833X_Gamma]] 作为基础亮度映射，手册建议优先使用 CGamma，不建议用 YGamma 强改色彩。
- DRC curve 和 bright 会影响亮度，但 `bright` 建议保持稳定。

## 4. 色彩调优
1. 先调 [[wiki/modules/FH833X_AWB|FH833X_AWB]]：白点、白框、色温框、统计阈值、场景权重。
2. 再调 [[wiki/modules/FH833X_CCM|FH833X_CCM]]：使用 24 色卡生成矩阵，并在线微调。
3. 最后看 [[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]] / [[wiki/modules/FH833X_CNR|FH833X_CNR]] / [[wiki/modules/FH833X_PURPLE|FH833X_PURPLE]] / [[wiki/modules/FH833X_FC|FH833X_FC]] / [[wiki/modules/FH833X_HLR|FH833X_HLR]] 等辅助模块。
4. 调 CCM 前确认 DRC 亮区 / 暗区饱和度没有抑制，CE、LCE、LC、CNR、Purple、FC、HLR 不要误伤颜色。

## 5. 对比度和通透性调优
- 推荐强 DRC、弱 Gamma 的思路：先用 DRC curve 让暗部看得见、亮部不过曝。
- 再通过 DRC 窗口滤波和细节增益加强局部对比。
- [[wiki/modules/FH833X_LCE|FH833X_LCE]] 用于进一步提升局部对比和通透性。
- [[wiki/modules/FH833X_YIE|FH833X_YIE]] / YC 的 contrast / brightness 通常保持 1 倍和 0，仅作为后期补偿。
- 各照度下需要分别调 gainMapping，避免白天通透、夜晚噪声颗粒过大。

## 6. 清晰度和噪声调优
- 清晰度和噪声必须联调，低照下不能单独追清晰度。
- 先确认光学成像：镜头干净、IRCut 正确、sensor 表面干净、无单边模糊、景深足够。
- 先在静止画面下定 APC 基础清晰度，再逐步调 NR3D、NR2D、[[wiki/modules/FH833X_YNR|FH833X_YNR]]、[[wiki/modules/FH833X_CNR|FH833X_CNR]]、[[wiki/modules/FH833X_DPC|FH833X_DPC]]。
- [[wiki/modules/FH833X_NR|FH833X_NR]] 要先标定 NR3D 运动 / 静止 coeff，再调 NR2D 联动。
- [[wiki/modules/FH833X_APC|FH833X_APC]] 要随 gain 降低低照锐化，避免噪声被放大。

## 7. WDR 模式专项流程
1. 确认 WDR 输入模式：三帧、两帧或 Sensor 合成。
2. 配置 [[wiki/modules/FH833X_WDR|FH833X_WDR]] mode 和曝光比，常用 16 倍但必须符合 sensor 能力。
3. 用 `wdr_merge_cfg` 单出长 / 中 / 短帧，确认每帧亮度、噪声和曝光实现。
4. 输出合成帧，手动 AE 下观察高亮随曝光变化是否闪烁。
5. 若高亮闪烁，调整短帧 / 中帧亮度矫正时序。
6. 按场景选择 merge 曲线，必要时自定义曲线。
7. WDR 下联调 DRC、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、NR 和 APC，尤其注意中 / 短帧噪声和运动伪影。
