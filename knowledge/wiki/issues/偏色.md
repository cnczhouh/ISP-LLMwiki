# 偏色

偏色是 ISP 调试中最常见的问题之一，表现为白色物体不再中性，或画面整体偏黄、偏绿、偏红、偏蓝。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO Semiconductor / Fullhan
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/SC360AT|SC360AT]] / [[wiki/platforms/SC361AT|SC361AT]] / [[wiki/platforms/GW5|GW5]] / [[wiki/platforms/FH833X|FH833X]]
- 模块：[[wiki/modules/ISX031_AWB|AWB]] / [[wiki/modules/ISX031_CCM|CCM]] / [[wiki/modules/ISX031_Gamma|Gamma]] / [[wiki/modules/ISX031_ATR|ATR]] / [[wiki/modules/ISX031_Pre White Balance|Pre White Balance]] / [[wiki/modules/SC121AT_AWB|SC121AT_AWB]] / [[wiki/modules/SC121AT_CCM|SC121AT_CCM]] / [[wiki/modules/SC360AT_AWB|SC360AT_AWB]] / [[wiki/modules/SC360AT_CCM|SC360AT_CCM]] / [[wiki/modules/SC360AT_LSC|SC360AT_LSC]] / [[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]] / [[wiki/modules/SC361AT_MWB|SC361AT_MWB]] / [[wiki/modules/GW5_AWB|GW5_AWB]] / [[wiki/modules/GW5_Color|GW5_Color]] / [[wiki/modules/GW5_LSC|GW5_LSC]] / [[wiki/modules/FH833X_AWB|FH833X_AWB]]
- 场景：白物体不再中性、肤色异常、室内外切换颜色异常
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]]、[[wiki/platforms/SC361AT|SC361AT]] 与 [[wiki/platforms/GW5|GW5]]，后续可继续扩展为跨平台偏色排查知识

## 现象描述
- 白墙、灰卡、白纸不再中性
- 肤色异常
- 室内外切换时颜色方向明显变化

## 排查顺序
1. 先确认 [[wiki/modules/ISX031_AE|AE]] 是否稳定，排除严重曝光干扰。
2. 检查 [[wiki/modules/ISX031_Pre White Balance|Pre White Balance]] 是否已正确完成。
3. 检查 [[wiki/modules/ISX031_AWB|AWB]] 窗口位置、场景模式、`Ratio/Aim/Cont` 是否合理。
4. 若 AWB 收敛正常但颜色仍偏，检查 [[wiki/modules/ISX031_CCM|CCM]]。
5. 若只是观感偏怪，回看 [[wiki/modules/ISX031_Gamma|Gamma]] / [[wiki/modules/ISX031_ATR|ATR]] 是否改变了层次和颜色印象。

## 典型判断依据
- 灰卡是否接近中性
- R/G、B/G 收敛趋势是否稳定
- 不同光源场景下偏色方向是否一致

## 平台差异入口
- 不同平台里偏色的主责模块可能不同：有的平台更依赖 [[wiki/modules/ISX031_AWB|AWB]] / [[wiki/modules/ISX031_CCM|CCM]]，有的平台前级补偿影响更大。
- 若后续接入新平台，优先补“相关平台页”和本页“调试案例 / 项目记录”，不要直接改写主干排查顺序。

- SC121AT：优先确认 [[wiki/modules/SC121AT_AWB|SC121AT_AWB]] 的 Calibration / Debug 流程是否正确，先用 Check_White_Pixel 标记确认白点是否被正确识别；若白点正确但白色仍偏，检查 B_Gain / R_Gain 是否被 Step1 的 Min / Max 限制，再看 [[wiki/modules/SC121AT_CCM|SC121AT_CCM]]、[[wiki/modules/SC121AT_Saturation|Saturation]] 等色彩风格调节。
- SC360AT：优先关联 [[wiki/modules/SC360AT_AWB|SC360AT_AWB]]、[[wiki/modules/SC360AT_CCM|SC360AT_CCM]]、[[wiki/modules/SC360AT_LSC|SC360AT_LSC]] 和 [[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]]。先确认 AWB Calibration Step1 / Step2、白点边界、`Abandon White Point` 和 `AWB_WP_WEIGHT` 是否正确；若灰卡中性但画面仍偏，再看高 / 中 / 低色温 CCM 插值、LSC 色彩阴影和局部饱和度策略。
- SC361AT：优先确认 [[wiki/modules/SC361AT_MWB|SC361AT_MWB]] 是否开启，以及 R / G / B gain 是否按平台预期回写；若外部 ISP 也在做 AWB，需要区分 sensor 端 MWB 与 ISP 端 AWB 的控制权。
- GW5：优先关联 [[wiki/modules/GW5_AWB|GW5_AWB]]、[[wiki/modules/GW5_Color|GW5_Color]] 和 [[wiki/modules/GW5_LSC|GW5_LSC]]。若灰卡不稳，先复核 AWB Mesh、Planckian / extra white point、EV-to-lux LUT 和 Night Mode；若白点正确但画面仍偏，检查 CCM、saturation、LSC chroma shading 与 chroma noise。
- FH833X：优先关联 [[wiki/modules/FH833X_AWB|FH833X_AWB]]、[[wiki/modules/FH833X_BLC|FH833X_BLC]]、[[wiki/modules/FH833X_LSC|FH833X_LSC]] 和 [[wiki/modules/FH833X_CCM|FH833X_CCM]]。先检查白点标定、白框是否覆盖 2000K~10000K 极限色温、色温框宽度、`statThl/statThh` 是否剔除过暗 / 过曝块；混合色温偏色重点看 `multiCTWeight0~11`、室外 `outWeight0~11` 和低照 `lowLightCTWeight0~11`。
