# 假边

假边是指边缘周围出现不真实的白边、黑边或双边缘，通常由锐化过强或前后级处理叠加造成。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO Semiconductor
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/SC360AT|SC360AT]] / [[wiki/platforms/SC361AT|SC361AT]] / [[wiki/platforms/GW5|GW5]]
- 模块：[[wiki/modules/ISX031_Sharpen|Sharpen]] / [[wiki/modules/ISX031_NR|NR]] / [[wiki/modules/SC360AT_Sharpness|SC360AT_Sharpness]] / [[wiki/modules/SC360AT_CAC|SC360AT_CAC]] / [[wiki/modules/SC360AT_HDR|SC360AT_HDR]] / [[wiki/modules/SC361AT_HDR|SC361AT_HDR]] / [[wiki/modules/GW5_GE|GW5_GE]] / [[wiki/modules/GW5_Demosaic|GW5_Demosaic]] / [[wiki/modules/GW5_SNR|GW5_SNR]]
- 场景：高对比边缘、文字边缘、白天高照场景、锐化过强后
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]]、[[wiki/platforms/SC361AT|SC361AT]] 与 [[wiki/platforms/GW5|GW5]]，后续可继续扩展为跨平台假边排查知识

## 现象表现
- 高对比边缘带白边或黑边
- 文字边缘发飘、双线化
- 轮廓看起来很“利”，但不真实

## 常见触发场景
- 锐化增益过高
- coring 过低导致噪声也参与锐化
- 降噪与锐化搭配失衡

## 可能相关模块
- [[wiki/modules/ISX031_Sharpen|Sharpen]]
- [[wiki/modules/ISX031_NR|NR]]

## 优先排查顺序
1. 优先检查 [[wiki/modules/ISX031_Sharpen|Sharpen]] 的边缘增强系数。
2. 检查 coring / 最大切割值是否过于激进。
3. 联查 [[wiki/modules/ISX031_NR|NR]] 是否先把纹理抹平，再被锐化拉出描边感。

## 关键观察项
- 文字与高对比边缘是否有白边黑边
- 人脸轮廓是否发硬
- 假边是否在高照和低照表现不同

## 常见误判
- 把“更锐”直接当成“更清晰”
- 只看静态单图，不看动态轮廓和真实纹理

## 平台差异入口
- 不同平台的假边可能落在锐化算法、降噪前后顺序和细节增强策略上。
- 相关平台页：[[wiki/platforms/ISX031|ISX031]]

- SC121AT：优先关联 HDR Combine Edge Tuning 与 [[wiki/modules/SC121AT_Sharpness|SC121AT_Sharpness]]。若假边出现在高反差或 HDR 合成边缘，先检查 HDR 边缘合成，再检查锐化强度和限制项。
- SC360AT：优先关联 [[wiki/modules/SC360AT_Sharpness|SC360AT_Sharpness]]、[[wiki/modules/SC360AT_CAC|SC360AT_CAC]] 和 [[wiki/modules/SC360AT_HDR|SC360AT_HDR]]。文字和建筑边缘发硬时先看 Sharpness 的边缘阈值、白 / 黑边控制和 gain 自适应；蓝紫边或高光边缘异常时看 CAC 的颜色、边缘、蓝亮度条件；若只在 HDR 高反差边界出现，回查 Combine 的 Blooming / Hist / Comps。
- SC361AT：优先关联 [[wiki/modules/SC361AT_HDR|SC361AT_HDR]]。若假边集中在高反差边缘或 HDR 过渡区，先看 HCG / LCG / OF / SE 合成边界与 PWL / LTM 压缩映射，再判断是否是后级锐化问题。
- GW5：优先关联 [[wiki/modules/GW5_GE|GW5_GE]]、[[wiki/modules/GW5_Demosaic|GW5_Demosaic]]、[[wiki/modules/GW5_SNR|GW5_SNR]] 和 [[wiki/modules/GW5_PFC|GW5_PFC]]。GE 应在 Demosaic 前确认，避免 Gr/Gb 不平衡被后级放大；若假边伴随 false color 或噪声增强，再联查 Demosaic 方向判断、sharpening 与 SNR。
- FH833X：优先关联 [[wiki/modules/FH833X_FC|FH833X_FC]]、[[wiki/modules/FH833X_PURPLE|FH833X_PURPLE]]、[[wiki/modules/FH833X_GB|FH833X_GB]] 和 [[wiki/modules/FH833X_APC|FH833X_APC]]。高频彩边先看 FC 的 `offset_t/offset_y/str`，高光边缘紫边看 PURPLE 的 Hue 阈值和 `Ydiffth`，GB / APC 过强也可能造成边缘异常或视觉坏点。
