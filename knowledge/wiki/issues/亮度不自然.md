# 亮度不自然

亮度不自然是指画面整体明暗关系虽然未必明显过曝或欠曝，但主观上会显得太平、太硬、太闷或层次分布不顺。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO / Fullhan
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/SC360AT|SC360AT]] / [[wiki/platforms/SC361AT|SC361AT]] / [[wiki/platforms/GW5|GW5]] / [[wiki/platforms/FH833X|FH833X]]
- 模块：[[wiki/modules/ISX031_AE|AE]] / [[wiki/modules/ISX031_ATR|ATR]] / [[wiki/modules/ISX031_Gamma|Gamma]] / [[wiki/modules/ISX031_Brightness Adjustment|Brightness Adjustment]] / [[wiki/modules/SC360AT_AE|SC360AT_AE]] / [[wiki/modules/SC360AT_HDR|SC360AT_HDR]] / [[wiki/modules/SC360AT_Gamma|SC360AT_Gamma]] / [[wiki/modules/SC360AT_Contrast|SC360AT_Contrast]] / [[wiki/modules/SC361AT_AEC_AGC|SC361AT_AEC_AGC]] / [[wiki/modules/SC361AT_PWL|SC361AT_PWL]] / [[wiki/modules/FH833X_AE|FH833X_AE]] / [[wiki/modules/FH833X_Gamma|FH833X_Gamma]] / [[wiki/modules/FH833X_DRC|FH833X_DRC]] / [[wiki/modules/FH833X_LCE|FH833X_LCE]]
- 场景：主观亮度风格调整、明暗层次异常、不同照度下观感不一致
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]]、[[wiki/platforms/SC361AT|SC361AT]]、[[wiki/platforms/GW5|GW5]] 与 [[wiki/platforms/FH833X|FH833X]]，后续可继续扩展为跨平台亮度观感排查知识

## 现象表现
- 画面不一定过曝或欠曝，但看起来别扭
- 中间调发闷、暗部发灰，或者高光顶得太硬
- 白天和夜晚的风格不统一

## 常见触发场景
- AE 落点正常，但 ATR / Gamma / 风格层影响观感
- 对比度和亮度基线联动失衡
- 不同照度档位亮度曲线切换不平顺

## 优先排查顺序
1. 先确认是真曝光问题还是主观亮度观感问题。
2. 若 RAW 落点就不对，先检查 [[wiki/modules/ISX031_AE|AE]]。
3. 若曝光落点正常但层次怪，联查 [[wiki/modules/ISX031_ATR|ATR]] 与 [[wiki/modules/ISX031_Gamma|Gamma]]。
4. 若只是最终风格偏平或偏硬，检查 [[wiki/modules/ISX031_Brightness Adjustment|Brightness Adjustment]]。

## 关键观察项
- 中间调是否自然
- 暗部是否松灰
- 高光是否早早顶死
- 不同照度档位风格是否一致

## 常见误判
- 把 tone mapping / Gamma 的问题误当成 AE 问题
- 只看单一场景讨好度，不看整机跨场景一致性

## 平台差异入口
- 不同平台的亮度观感问题可能落在曝光控制、局部动态范围压缩、Gamma 或主观风格层。
- 若后续接入新平台，优先补该平台的亮度映射链路与主观风格入口，再扩展到问题页。
- 相关平台页：[[wiki/platforms/ISX031|ISX031]]

- SC121AT：优先关联 [[wiki/modules/SC121AT_AE|SC121AT_AE]]、[[wiki/modules/SC121AT_HDR|SC121AT_HDR]]、[[wiki/modules/SC121AT_Gamma|SC121AT_Gamma]]、[[wiki/modules/SC121AT_Contrast|Contrast]] 与 Tonemapping。若曝光落点正常但观感不顺，应重点检查 HDR 压缩、Gamma 和对比度映射，而不是只调 AE 目标。

- SC360AT：优先关联 [[wiki/modules/SC360AT_AE|SC360AT_AE]]、[[wiki/modules/SC360AT_HDR|SC360AT_HDR]]、[[wiki/modules/SC360AT_Gamma|SC360AT_Gamma]] 和 [[wiki/modules/SC360AT_Contrast|SC360AT_Contrast]]。若四帧曝光 Mean 和 `USE_Exp.Mean` 正常但观感不顺，重点检查 Tonemapping 的 Dynamic Range / Global Gamma / Local Tonemapping / Hist EQ、GammaA/B/C 与 Contrast 的 Low Level / Stretch / gain 自适应，而不是只改 AE target。

- SC361AT：优先关联 [[wiki/modules/SC361AT_AEC_AGC|SC361AT_AEC_AGC]]、[[wiki/modules/SC361AT_HDR|SC361AT_HDR]] 和 [[wiki/modules/SC361AT_PWL|SC361AT_PWL]]。若整体亮度落点正常但中间调不顺、高光压得硬或暗部浮起，重点检查 HDR 分支曝光 / gain 约束和 PWL 节点、slope 是否平滑。

- GW5：优先区分 [[wiki/modules/GW5_AE|GW5_AE]] 落点、Gamma 全局曲线和 [[wiki/modules/GW5_LTM|GW5_LTM]] 局部动态范围压缩。若 LTM strength、asymmetry LUT、variance 或 slope 限制不合适，可能造成暗部发灰、高光发暗、局部 halo 或中间调不自然。

- FH833X：优先关联 [[wiki/modules/FH833X_AE|FH833X_AE]]、[[wiki/modules/FH833X_Gamma|FH833X_Gamma]]、[[wiki/modules/FH833X_DRC|FH833X_DRC]]、[[wiki/modules/FH833X_LCE|FH833X_LCE]] 和 [[wiki/modules/FH833X_YIE|FH833X_YIE]]。若曝光落点正常但亮度层次不顺，重点区分 AE target、CGamma / YGamma、DRC curve / bright、LCE 局部对比和 YIE 后级亮度 / 对比度补偿。
