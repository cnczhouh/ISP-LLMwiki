# 固定亮点暗点

固定亮点暗点是指画面固定位置出现亮点、暗点、热噪点或异常像素，通常和 DPC、坏点表、温度、gain 和 sensor 本体状态相关。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO
- 平台：跨平台
- 模块：DPC / Bad Pixel Correction / Gain / Temperature
- 场景：固定亮点、固定暗点、高温热噪点、低照异常点
- 适用范围：跨平台

## 现象表现
- 固定位置出现白点、黑点或彩点。
- 高 gain 或高温下异常点数量增加。
- 静态图像中点位稳定，视频中不随纹理运动。
- DPC 过强时异常点消失，但星点或小纹理也可能被误伤。

## 优先排查顺序
1. 区分随机噪声和固定坏点，固定点应在多帧中位置稳定。
2. 分别在常温 / 高温、低 gain / 高 gain 下观察点位数量变化。
3. 检查 DPC / BPC 是否开启，以及亮点、暗点通道是否分别生效。
4. 若平台支持坏点表或 OTP / EEPROM 校正，检查坏点表是否加载。
5. 若 DPC 后细节被误伤，联查 [[wiki/issues/细节损失|细节损失]]。

## 常见处理方向
- 开启或调整亮点 / 暗点校正。
- 高温或高 gain 场景单独验证 DPC 强度。
- 对真实坏点和热噪点分开判断，不要只用单一常温图确认。
- 避免 DPC 阈值过强导致星点、小纹理和细线被抹掉。

## 平台差异入口
- SC361AT：优先关联 [[wiki/modules/SC361AT_DPC|SC361AT_DPC]]，检查 white pixel / black pixel cancellation 开关。
- GW5：优先关联 [[wiki/modules/GW5_DPC|GW5_DPC]]，确认 DPC 在 RAW 前级链路中的位置和强度。
- ISX031：优先关联 [[wiki/modules/ISX031_Spot Pixel Compensation|Spot Pixel Compensation]]。
- FH833X：优先关联 [[wiki/modules/FH833X_DPC|FH833X_DPC]]。正常照度先用单点坏点校正，低照可考虑多点模式；调小白 / 黑点阈值会增强去坏点，但会增加细节损失，WDR 中短帧坏点可用 `mfStrCtrl` 单独加强。
