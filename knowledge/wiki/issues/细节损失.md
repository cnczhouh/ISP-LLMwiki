# 细节损失

细节损失是指纹理、边缘或微小结构被过度平滑或压掉，导致画面发肉、发糊、缺少真实质感。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO Semiconductor
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/SC360AT|SC360AT]] / [[wiki/platforms/SC361AT|SC361AT]] / [[wiki/platforms/GW5|GW5]]
- 模块：[[wiki/modules/ISX031_NR|NR]] / [[wiki/modules/ISX031_Sharpen|Sharpen]] / [[wiki/modules/SC121AT_NR|SC121AT_NR]] / [[wiki/modules/SC121AT_Sharpness|SC121AT_Sharpness]] / [[wiki/modules/SC360AT_NR|SC360AT_NR]] / [[wiki/modules/SC360AT_Sharpness|SC360AT_Sharpness]] / [[wiki/modules/SC361AT_DPC|SC361AT_DPC]] / [[wiki/modules/GW5_SNR|GW5_SNR]] / [[wiki/modules/GW5_CNR|GW5_CNR]] / [[wiki/modules/GW5_Demosaic|GW5_Demosaic]]
- 场景：高降噪强度、低照场景、纹理丰富场景、细节观感不足
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]]、[[wiki/platforms/SC361AT|SC361AT]] 与 [[wiki/platforms/GW5|GW5]]，后续可继续扩展为跨平台细节损失排查知识

## 现象表现
- 衣物、头发、草地等纹理被抹平
- 画面干净但缺少真实细节
- 边缘不一定有假边，但整体显肉

## 常见触发场景
- NR 过强
- coring 太高
- 低照度下为了压噪声牺牲了真实纹理

## 可能相关模块
- [[wiki/modules/ISX031_NR|NR]]
- [[wiki/modules/ISX031_Sharpen|Sharpen]]

## 优先排查顺序
1. 先检查 [[wiki/modules/ISX031_NR|NR]] 是否压得过重。
2. 再检查 [[wiki/modules/ISX031_Sharpen|Sharpen]] 的 coring 与高频提取策略。
3. 在高照和低照分别确认问题是否一致，避免只照顾单一场景。

## 关键观察项
- 细纹理是否还存在
- 暗部纹理是否被抹平
- 文字和边缘是否既不锐也不真

## 常见误判
- 只以噪声干净为目标，忽略真实纹理丢失
- 把细节损失误当成单纯对焦问题

## 平台差异入口
- 不同平台的细节损失可能来自 RAW 降噪、色度降噪、锐化 coring 或多帧处理。
- 相关平台页：[[wiki/platforms/ISX031|ISX031]]

- SC121AT：优先关联 [[wiki/modules/SC121AT_NR|SC121AT_NR]]、[[wiki/modules/SC121AT_Sharpness|SC121AT_Sharpness]] 与 Tonemapping。若纹理被抹平，先确认 NR 是否过强，再看 Sharpness 是否只是把被抹掉的边缘硬拉回来。
- SC360AT：优先关联 [[wiki/modules/SC360AT_NR|SC360AT_NR]] 和 [[wiki/modules/SC360AT_Sharpness|SC360AT_Sharpness]]。先检查 DNS13 / DNS23 的 gain node、边缘判定和 denoise strength 是否把纹理当噪声压掉，再看 DnsBack / add back 是否不足；若后级 Sharpness 只是把残留边缘拉硬，说明应回到 NR 分支重新平衡而不是继续加锐。
- SC361AT：优先关联 [[wiki/modules/SC361AT_DPC|SC361AT_DPC]] 和 [[wiki/modules/SC361AT_PWL|SC361AT_PWL]]。若细小纹理、星点或固定微小亮点被抹掉，先确认 DPC 是否过强；若层次被压平，再检查 PWL 节点和 slope 是否过硬。
- GW5：优先关联 [[wiki/modules/GW5_SNR|GW5_SNR]]、[[wiki/modules/GW5_CNR|GW5_CNR]] 和 [[wiki/modules/GW5_Demosaic|GW5_Demosaic]]。若纹理损失集中在低照或高 gain，先分开检查 RAW SNR、色度 CNR 和 Demosaic / sharpening，避免用后级锐化硬补已被降噪抹掉的细节。
- FH833X：优先关联 [[wiki/modules/FH833X_DPC|FH833X_DPC]]、[[wiki/modules/FH833X_GB|FH833X_GB]]、[[wiki/modules/FH833X_YNR|FH833X_YNR]] 和 [[wiki/modules/FH833X_SHARP_NOISE|FH833X_SHARP_NOISE]]。DPC / GB / YNR 过强都会损伤细节，低照需在坏点、格子纹、噪声和清晰度之间权衡。
