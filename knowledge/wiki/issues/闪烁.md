# 闪烁

闪烁是指画面亮度随时间周期性波动，或出现与工频光源相关的明暗条纹，常见于室内人工照明场景。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO Semiconductor / Fullhan
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/GW5|GW5]] / [[wiki/platforms/FH833X|FH833X]]
- 模块：[[wiki/modules/ISX031_AE|AE]] / [[wiki/modules/GW5_AE|GW5_AE]] / [[wiki/modules/GW5_AFD|GW5_AFD]] / [[wiki/modules/FH833X_AE|FH833X_AE]] / [[wiki/modules/FH833X_WDR|FH833X_WDR]]
- 场景：50Hz/60Hz 工频灯光、室内照明、快门与电源频率不匹配
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]] 与 [[wiki/platforms/GW5|GW5]]，后续可继续扩展为跨平台闪烁排查知识

## 现象表现
- 画面亮度周期性忽明忽暗
- 室内灯光下出现横条或亮度跳变
- 录像中比静态图更容易看到闪烁

## 常见触发场景
- 50Hz / 60Hz 工频灯具
- 快门时间未与工频同步
- AE 在抗闪烁模式下切换策略不稳定

## 可能相关模块
- [[wiki/modules/ISX031_AE|AE]]

## 优先排查顺序
1. 先确认场景是否为工频灯光环境。
2. 检查 [[wiki/modules/ISX031_AE|AE]] 的 50/60Hz 无闪烁设置是否正确。
3. 检查快门时间是否落在合理的防闪烁档位。
4. 若开启防闪烁后亮度又异常，联查曝光目标与增益取舍。

## 关键观察项
- 是否有横条
- 是否周期性亮度跳变
- 快门时间是否锁在合理档位
- 开启无闪烁后暗场增益是否明显升高

## 常见误判
- 把 LED 光源频闪误判成普通曝光抖动
- 只盯亮度，不看快门是否已经脱离工频约束

## 平台差异入口
- 不同平台的抗闪烁能力可能落在 AE 策略、快门约束和电源频率检测逻辑。
- 若后续接入新平台，优先补该平台的无闪烁策略与快门档位规则，再扩展到问题页。
- 相关平台页：[[wiki/platforms/ISX031|ISX031]]

- SC121AT：优先关联 [[wiki/modules/SC121AT_AE|SC121AT_AE]] 的 Antiflicker 设置、曝光行数与光源频率。室内灯光下应先确认 50Hz / 60Hz 配置，再检查 AE 是否因防闪烁约束导致增益升高或亮度波动。
- SC360AT：优先关联 [[wiki/modules/SC360AT_AE|SC360AT_AE]] 的 FAD。SC360AT HCG / LCG / S 使用相同曝光，VS 帧不支持 flicker 抑制，通常重点打开 HCG banding 抑制，并检查 `Band_0/Band_1` 的 50 / 60Hz 预设曝光。
- GW5：优先关联 [[wiki/modules/GW5_AFD|GW5_AFD]] 与 [[wiki/modules/GW5_AE|GW5_AE]]。AFD 通常在其他模块完成后验证；若默认 AFD 无法检测 50Hz 且仍有频闪，可逐步降低 `afd.area_power_threshold`、`afd.peak_average_ratio_threshold` 和 `afd.detection_threshold_50Hz`，同时确认 AE 的 `power_line_frequency` 与曝光时间约束。
- FH833X：先关闭 DRC、LCE 等亮度影响模块判断是否 AE 闪烁；若是 AE 问题，重点检查 [[wiki/modules/FH833X_AE|FH833X_AE]] 的 intt / sensor gain / ISP gain 延迟、精细快门 / 精细增益内插和抗闪 `antiFlickerMode/frequency/stabZone`。WDR 高亮闪烁还要检查 [[wiki/modules/FH833X_WDR|FH833X_WDR]] 的短帧亮度矫正时序。
