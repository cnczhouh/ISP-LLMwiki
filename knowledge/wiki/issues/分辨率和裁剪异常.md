# 分辨率和裁剪异常

分辨率和裁剪异常是指输出图像尺寸、有效区域、crop offset、缩放比例、黑边或视场范围与预期不一致。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家
- 平台：跨平台
- 模块：Window / Crop / Scale / Output Size / ISP Pipeline
- 场景：分辨率配置、输出窗口、缩放、视场检查、接收端解析
- 适用范围：跨平台

## 现象表现
- 输出尺寸和配置不一致。
- 画面有黑边、偏移、裁掉一侧或有效区域不居中。
- 图像被拉伸、压缩或缩放比例不对。
- HDR / ISP 模式切换后尺寸或视场变化。

## 优先排查顺序
1. 确认 sensor active array、输出窗口、crop 和 scale 配置。
2. 检查接收端 width / height / stride 是否按实际输出设置。
3. 确认是否开启 ISP scale down、digital crop、binning 或 subsampling。
4. HDR 多路输出时确认各路尺寸、VC 和接收端拼接方式一致。
5. 若画面同时花屏，先联查 [[wiki/issues/无图花屏|无图花屏]] 和 [[wiki/issues/输出格式错误|输出格式错误]]。

## 常见处理方向
- 点亮阶段先使用 datasheet 推荐的标准分辨率。
- 修改 crop / scale 后同步更新接收端尺寸和 stride。
- 标定类功能如 LSC 需要确认 raw-full size 或标定要求的输出尺寸。
- 车载场景要结合镜头视场、畸变校正和最终显示区域一起确认。
