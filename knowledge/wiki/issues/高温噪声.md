# 高温噪声

高温噪声是指 sensor 或模组温度升高后，暗电流、热噪点、固定亮点、暗部颗粒或画面发脏明显加重的问题。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens
- 平台：跨平台
- 模块：Temperature / HTEMP / NR / DPC / Gain
- 场景：高温老化、车载暴晒、长时间运行、低照高 gain
- 适用范围：跨平台

## 现象表现
- 温度升高后暗部噪声明显增加。
- 固定亮点、热噪点数量变多。
- 同一曝光 / gain 下，高温画面比常温更脏。
- 高温下 DPC 或 NR 不足，或补偿过强导致细节损失。

## 优先排查顺序
1. 记录温度、曝光、gain 和帧率，避免把高 gain 误当成纯温度问题。
2. 对比常温 / 高温暗场和低照实景，观察噪声类型变化。
3. 检查温度补偿、HTEMP、DPC、NR 是否随温度或 gain 生效。
4. 若固定点变多，联查 [[wiki/issues/固定亮点暗点|固定亮点暗点]]。
5. 若高温下细节明显变差，检查 NR / DPC 是否过度补偿。

## 常见处理方向
- 为高温场景单独验证 NR 和 DPC 强度。
- 控制低照高温下的 gain 上限，避免噪声被过度放大。
- 若平台有温度传感器或 HTEMP 模块，确认温度读取和补偿曲线正确。
- 高温测试要同时看静态暗场、低照实景和动态场景。
