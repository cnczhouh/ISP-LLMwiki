# 锐化过强

锐化过强是指边缘增强超过自然观感，出现白边、黑边、halo、噪声被增强或画面整体发硬的问题。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO / Fullhan
- 平台：跨平台 / [[wiki/platforms/SC360AT|SC360AT]]
- 模块：Sharpen / Demosaic / NR / CAC / LTM / APC / [[wiki/modules/SC360AT_Sharpness|SC360AT_Sharpness]] / [[wiki/modules/SC360AT_NR|SC360AT_NR]] / [[wiki/modules/SC360AT_CAC|SC360AT_CAC]]
- 场景：边缘发硬、白边黑边、噪声被锐化、文字边缘异常
- 适用范围：跨平台

## 现象表现
- 物体边缘出现明显白边或黑边。
- 文字、树枝、建筑边缘发硬，不自然。
- 暗部噪声、草地纹理或压缩噪声被一起增强。
- 高反差边缘可能伴随彩边或 halo。

## 优先排查顺序
1. 先确认是否是对焦或镜头解析力问题，不要用锐化硬补失焦。
2. 检查 NR 是否先把真实纹理抹掉，再被 Sharpen 拉出假细节。
3. 降低边缘增强强度，观察白边 / 黑边是否缓解。
4. 检查 coring / threshold，避免噪声被当作细节增强。
5. 高反差边缘若伴随彩边，联查 [[wiki/issues/假边|假边]] 和 CAC / Demosaic。

## 常见处理方向
- 先定 NR，再定 Sharpen，避免“强降噪 + 强锐化”的塑料感。
- 高 gain / 低照档位降低锐化强度，避免噪声被放大。
- 对正负边缘分别限制，避免白边或黑边过强。
- 局部 halo 明显时，联查 LTM / HDR 合成边缘。

- ISX031：优先关联 [[wiki/modules/ISX031_Sharpen|Sharpen]] 和 [[wiki/modules/ISX031_NR|NR]]。
- GW5：优先关联 [[wiki/modules/GW5_Demosaic|GW5_Demosaic]]、[[wiki/modules/GW5_SNR|GW5_SNR]] 和 [[wiki/modules/GW5_LTM|GW5_LTM]]。
- FH833X：优先关联 [[wiki/modules/FH833X_APC|FH833X_APC]]、[[wiki/modules/FH833X_NR|FH833X_NR]]、[[wiki/modules/FH833X_YNR|FH833X_YNR]]、[[wiki/modules/FH833X_SHARP_NOISE|FH833X_SHARP_NOISE]] 和 [[wiki/modules/FH833X_DRC|FH833X_DRC]]。重点检查 `pgain/ngain`、`posDeStr/negDeStr`、`posEsStr/negEsStr`、强边抑制 `locctrlSupsteady/locctrlSupstep`，以及低照亮度锐化强度是否把暗部白点和噪声锐化出来。
