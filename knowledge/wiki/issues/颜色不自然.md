# 颜色不自然

颜色不自然是指画面颜色虽然不一定明显偏色，但会表现出假艳、发闷、发脏、发塑料感或不符合主观预期的颜色观感。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO Semiconductor / Fullhan
- 平台：[[wiki/platforms/ISX031|ISX031]] / [[wiki/platforms/SC121AT|SC121AT]] / [[wiki/platforms/GW5|GW5]] / [[wiki/platforms/FH833X|FH833X]]
- 模块：[[wiki/modules/ISX031_CCM|CCM]] / [[wiki/modules/ISX031_Color Adjustment|Color Adjustment]] / [[wiki/modules/ISX031_Chroma Suppress|Chroma Suppress]] / [[wiki/modules/ISX031_Gamma|Gamma]] / [[wiki/modules/GW5_AWB|GW5_AWB]] / [[wiki/modules/GW5_Color|GW5_Color]] / [[wiki/modules/GW5_Gamma|GW5_Gamma]] / [[wiki/modules/GW5_LTM|GW5_LTM]] / [[wiki/modules/FH833X_AWB|FH833X_AWB]] / [[wiki/modules/FH833X_CCM|FH833X_CCM]] / [[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]] / [[wiki/modules/FH833X_FC|FH833X_FC]] / [[wiki/modules/FH833X_HLR|FH833X_HLR]]
- 场景：色彩风格调整、饱和度调整后、低照颜色观感异常、肤色不讨好
- 适用范围：当前已覆盖 [[wiki/platforms/ISX031|ISX031]]、[[wiki/platforms/SC121AT|SC121AT]]、[[wiki/platforms/GW5|GW5]] 与 [[wiki/platforms/FH833X|FH833X]]，后续可继续扩展为跨平台颜色观感排查知识

## 现象表现
- 颜色不脏不偏，但看起来“假”
- 草地、天空、红色物体过艳或发闷
- 暗部彩色纹理显脏，或者颜色被压得太死

## 常见触发场景
- 全局饱和度拉得过高或过低
- 局部色调调整过猛
- 低照度下色度抑制或降噪太重
- Gamma / 对比度风格让颜色观感一起变怪

## 优先排查顺序
1. 先区分是颜色方向错，还是颜色风格不自然。
2. 若是全局颜色关系怪，优先检查 [[wiki/modules/ISX031_CCM|CCM]]。
3. 若是饱和度、色调或局部颜色味道异常，检查 [[wiki/modules/ISX031_Color Adjustment|Color Adjustment]]。
4. 若低照下颜色发灰或发脏，联查 [[wiki/modules/ISX031_Chroma Suppress|Chroma Suppress]] 与 [[wiki/modules/ISX031_NR|NR]]。
5. 若只是层次变化导致“看起来颜色怪”，再回看 [[wiki/modules/ISX031_Gamma|Gamma]] 与 [[wiki/modules/ISX031_ATR|ATR]]。

## 关键观察项
- 肤色是否自然
- 草地、天空、红色物体是否假艳
- 暗部彩色纹理是否发脏
- 不同照度和光源下风格是否一致

## 常见误判
- 把主观风格问题误当成 AWB 偏色
- 一味提升饱和度，试图掩盖 CCM 或 Gamma 的根因

## 平台差异入口
- 不同平台里，颜色不自然可能来自矩阵、色调增强、色度抑制或 tone mapping 联动。
- 若后续接入新平台，优先补该平台的色彩风格入口与低照色彩链路，再扩展到问题页。
- 相关平台页：[[wiki/platforms/ISX031|ISX031]]

- SC121AT：优先关联 [[wiki/modules/SC121AT_Saturation|SC121AT_Saturation]]、CCM、Contrast、[[wiki/modules/SC121AT_Gamma|SC121AT_Gamma]] 和 [[wiki/modules/SC121AT_AWB|SC121AT_AWB]]。先确认 AWB 白点识别没有问题，再判断是饱和度、矩阵还是亮度映射导致颜色发假。
- GW5：优先关联 [[wiki/modules/GW5_AWB|GW5_AWB]]、[[wiki/modules/GW5_Color|GW5_Color]]、[[wiki/modules/GW5_Gamma|GW5_Gamma]] 和 [[wiki/modules/GW5_LTM|GW5_LTM]]。先确认 AWB 白点和 CCM / Saturation 标定，再判断 Gamma 全局亮度和 LTM 局部 tone mapping 是否让颜色显得发灰、发闷或过硬。
- FH833X：优先关联 [[wiki/modules/FH833X_AWB|FH833X_AWB]]、[[wiki/modules/FH833X_CCM|FH833X_CCM]]、[[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]]、[[wiki/modules/FH833X_CNR|FH833X_CNR]]、[[wiki/modules/FH833X_FC|FH833X_FC]] 和 [[wiki/modules/FH833X_HLR|FH833X_HLR]]。先确认 AWB / CCM 基础色准，再看 CE / LC 是否过度增强，CNR 是否压灰颜色，FC 是否把正常高频颜色去灰，HLR 是否在高光区域恢复过强或不足。
