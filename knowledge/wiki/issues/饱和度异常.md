# 饱和度异常

饱和度异常是指画面颜色过浓、过淡，或局部颜色在高亮、低照、特定色相场景下出现溢出、发脏、压不住的问题。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO
- 平台：跨平台 / [[wiki/platforms/SC360AT|SC360AT]]
- 模块：Saturation / CCM / AWB / Gamma / Tone Mapping / [[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]] / [[wiki/modules/SC360AT_CCM|SC360AT_CCM]] / [[wiki/modules/SC360AT_AWB|SC360AT_AWB]]
- 场景：颜色过浓、颜色过淡、低照脏色、蓝天异常、红色场景溢出
- 适用范围：跨平台

## 现象表现
- 整体颜色太浓，画面讨好但不真实。
- 整体颜色太淡，画面发灰、缺少活力。
- 低照下彩噪或脏色被饱和度放大。
- 蓝天、车灯、尾灯、交通标志等局部颜色异常突出。

## 优先排查顺序
1. 先确认 [[wiki/issues/偏色|偏色]] 是否已经解决，白点不准时不要先调饱和度。
2. 检查 CCM 是否导致色相和色彩强度整体偏离。
3. 检查全局 Saturation 是否过高或过低。
4. 低照场景检查饱和度随 gain 衰减是否合理。
5. 特定颜色异常时，再检查蓝色、红色或局部亮度自适应相关参数。

## 常见处理方向
- 白点和 CCM 正常后，再调整全局饱和度风格。
- 低照下适当降低饱和度，避免彩噪显著。
- 高亮彩色区域异常时，优先做局部颜色抑制，不要用全局降饱和牺牲整图颜色。
- 车载场景要验证红色交通标志、尾灯和 LED 屏，避免局部压饱和过度。

## 平台差异入口
- SC121AT：优先关联 [[wiki/modules/SC121AT_Saturation|SC121AT_Saturation]]、[[wiki/modules/SC121AT_CCM|SC121AT_CCM]] 和 [[wiki/modules/SC121AT_AWB|SC121AT_AWB]]。低照脏色看 Bright Adaptive，蓝色 / 红色局部异常分别看 Blue Pixel 和 Red Scene。
- SC360AT：优先关联 [[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]]、[[wiki/modules/SC360AT_CCM|SC360AT_CCM]] 和 [[wiki/modules/SC360AT_AWB|SC360AT_AWB]]。先确认 AWB 和 CCM 色相基础正确，再看 gain 自适应降饱和、局部亮度降饱和、Blue Pixel 和 Traffic Light 增强；蓝天、尾灯和交通标志异常时优先调局部策略，不要直接全局降饱和。
- ISX031：优先关联 [[wiki/modules/ISX031_CCM|CCM]]、[[wiki/modules/ISX031_Chroma Suppress|Chroma Suppress]] 和 [[wiki/modules/ISX031_usr_adj|useradj / usr_adj]]。
- GW5：优先关联 [[wiki/modules/GW5_Color|GW5_Color]]、[[wiki/modules/GW5_AWB|GW5_AWB]] 和 [[wiki/modules/GW5_CNR|GW5_CNR]]。
- FH833X：优先关联 [[wiki/modules/FH833X_CCM|FH833X_CCM]]、[[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]] 和 [[wiki/modules/FH833X_CNR|FH833X_CNR]]。主饱和度优先由 CCM 保证，CE 只做辅助增强 / 抑制；LC 仅用于单色相小幅微调，避免过渡不平滑。
