# 色噪

色噪是指暗部、纯色区域或高 gain 场景中出现彩色斑点、彩色块、色度闪动或 chroma blotch 的问题。

## 页面属性
- 类型：通用问题
- 厂家：跨厂家 / Sony / SmartSens / GEO
- 平台：跨平台 / [[wiki/platforms/SC360AT|SC360AT]]
- 模块：CNR / NR / AWB / Saturation / Gain / [[wiki/modules/SC360AT_NR|SC360AT_NR]] / [[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]]
- 场景：低照、高 gain、暗部纯色区域、色块抖动
- 适用范围：跨平台

## 现象表现
- 暗部有红绿蓝彩点或彩色块。
- 灰墙、天空、车身等纯色区域颜色不干净。
- 视频中颜色噪声随时间闪动。
- 提高饱和度后色噪明显变严重。

## 优先排查顺序
1. 先确认曝光和 gain 是否过高，色噪通常随 gain 明显上升。
2. 区分亮度噪声和色度噪声，避免只调亮度 NR。
3. 检查 CNR / chroma NR 强度和随 gain 的自适应曲线。
4. 检查 Saturation 是否在低照下过高，导致彩噪被放大。
5. 若色噪带固定空间分布，联查 LSC / shading / 黑位补偿。

## 常见处理方向
- 高 gain 档位提高 CNR，同时保留必要颜色细节。
- 低照下降低 Saturation 或增加暗处降饱和。
- 避免过强 CNR 把真实低饱和颜色压成灰。
- 联合 AE gain 上限、NR、Saturation 做低照整体平衡。

## 平台差异入口
- SC121AT：优先关联 [[wiki/modules/SC121AT_NR|SC121AT_NR]] 和 [[wiki/modules/SC121AT_Saturation|SC121AT_Saturation]]，低照色噪明显时检查 Bright Adaptive 与 gain 自适应降饱和。
- SC360AT：优先关联 [[wiki/modules/SC360AT_NR|SC360AT_NR]]、[[wiki/modules/SC360AT_Saturation|SC360AT_Saturation]] 和 [[wiki/modules/SC360AT_AWB|SC360AT_AWB]]。先看 DNS13 / DNS23 中 BR 通道强度是否相对 G 通道不足或过强，BR 通常需比 G 更强以压制色边和色噪；低照色噪还要检查 gain 自适应降饱和、Blue Pixel / Traffic Light 等局部颜色逻辑是否把彩噪或特定色相放大。
- ISX031：优先关联 [[wiki/modules/ISX031_NR|NR]]、[[wiki/modules/ISX031_Chroma Suppress|Chroma Suppress]] 和 [[wiki/modules/ISX031_AE|AE]]。
- GW5：优先关联 [[wiki/modules/GW5_CNR|GW5_CNR]]、[[wiki/modules/GW5_SNR|GW5_SNR]] 和 [[wiki/modules/GW5_LTM|GW5_LTM]]。
- FH833X：优先关联 [[wiki/modules/FH833X_CNR|FH833X_CNR]]、[[wiki/modules/FH833X_CCM|FH833X_CCM]] 和 [[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]]。先用 CNR 控制色噪并通过 `col_thl/col_thh` 保护正常颜色；低照色噪还要检查 ACR 是否需要弱化 CCM，以及 CE 饱和度是否过强。
