# ISX031

Sony CMOS 图像传感器平台；当前知识节点聚焦于图像调优、寄存器配置、启动流程和图像质量相关功能。

## 规格参数总结
- 类型：Sony 3.01MP CMOS Image Sensor System-on-Chip，面向车载彩色摄像头
- 光学尺寸：Type 1/2.42，diagonal 7.45 mm
- 有效像素阵列：1937(H) x 1553(V)，约 3.01M pixel
- 推荐记录像素：1920(H) x 1536(V)，约 2.95M pixel
- 像素尺寸：3.0 μm(H) x 3.0 μm(V)
- 输出接口：MIPI CSI-2 serial data output，支持 4-lane / 2-lane
- 输出格式：YCbCr format
- 外部通信：I2C communication
- 输入时钟：OSCI 18–30 MHz，推荐 18 / 24 / 30 MHz
- 供电：analog 3.3 V、interface 1.8 V、digital 1.1 V
- 工作温度：Topr -40°C ~ +105°C；Tj -40°C ~ +125°C
- 封装尺寸：7.328 mm(H) x 6.266 mm(V) with scribe line；7.200 mm(H) x 6.138 mm(V) without scribe line
- 车载与可靠性：AEC-Q100 Grade 2 qualified；ESD HBM Grade 2 Classification 2，CDM Grade 2 Classification C2a
- HDR 能力：支持 sub-pixel method、internal combining、motion artifact free 和 LED flicker mitigation
- 图像处理能力重点：资料明确覆盖 [[wiki/modules/ISX031_HDR|HDR]]、[[wiki/modules/ISX031_AE|AE]]、[[wiki/modules/ISX031_AWB|AWB]]、[[wiki/modules/ISX031_ATR|ATR]]、[[wiki/modules/ISX031_Gamma|Gamma]]、[[wiki/modules/ISX031_NR|NR]]、[[wiki/modules/ISX031_CCM|CCM]]、[[wiki/modules/ISX031_Sharpen|Sharpen]] 以及多类前级补偿模块
- 调试资料特点：既有底层硬件/启动资料，也有面向图像质量的调参路径，适合把“平台规格”和“图像调试方法”放在同一知识体系下整理

## 规格参数阅读重点
- 看 datasheet 时，优先整理：分辨率、像素尺寸、输出位宽、帧率、供电、接口时序、工作温度这类硬规格
- 看 application note 时，优先整理：启动方式、I2C 地址/访问方式、MIPI CSI-2 输出配置、NOR Flash、驱动模式
- 看 image tuning manual 时，优先整理：各 ISP 模块作用、调试顺序、关键状态量、关键寄存器组
- 现阶段 wiki 已更完整覆盖“调试能力与流程”，后续若要补“硬规格表”，建议直接从 `[[raw/ISX031-AATH5-W1_datasheet_E_E23406_COB.pdf]]` 单独抽表补充

## 调试视角下的理解
- ISX031 不适合只当作“一个 sensor 型号”去记，更适合作为“带完整图像处理与调优入口的平台”去整理
- 真正常用的信息会分散在 datasheet、应用说明和调优手册中：datasheet 解决能不能接、怎么起，application note 解决怎么配，tuning manual 解决怎么把图调好
- 所以后续整理规格参数时，最好分成两层：一层是硬件规格，一层是图像调试能力，不要混成单一表格
