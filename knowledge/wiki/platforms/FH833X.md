# FH833X

Fullhan / 富瀚微 FH833X ISP 平台；当前知识节点基于《FH833X_效果调试手册-v2.0.4》，用于整理线性模式、WDR 模式、AE / AWB / WDR / DRC / NR / APC 等图像效果调试入口。

## 平台属性
- 类型：平台
- 主类型：ISP
- 附加属性：线性模式 / WDR / 三帧合成 / 两帧合成 / Sensor 合成 / 车载调试
- 厂家：Fullhan / 富瀚微
- 平台：[[wiki/platforms/FH833X|FH833X]]
- 场景：ISP 图像效果调试、sensor 接入、线性模式调优、WDR 模式调优、常见问题排查
- 适用范围：指定平台

## 平台定位
- FH833X 是 Fullhan 平台 ISP，手册明确适用于 FH833X。
- 平台支持线性模式和 WDR 模式：线性包含普通模式和 RGBir，WDR 包含三帧 / 两帧合成和 Sensor 合成。
- 手册主线是从 sensor 接入与标定开始，再按亮度、色彩、对比度和通透性、清晰度和噪声控制几个维度进行图像质量调优。

## ISP 模块概览
- 前级与基础：Color Bar、Crop、BLC0/1/2、DPC、GB、WB0/1/3、LSC、DG。
- WDR 与动态范围：WDR、LUT12、DRC。
- 降噪与细节：NR3D、NR2D、YNR、CNR、APC、CFA、FC。
- 色彩与观感：AWB、CCM、GAMMA、LCE、YIE、PURPLE、CHROMA、CIE、Local Chroma。
- 几何与输出：HMIR、LDC、RGB2YC。

## 调试主线
1. Sensor 接入：确认基本通路、帧率、分辨率、AE 控制曝光 / gain、亮暗态出图。
2. Sensor 与镜头标定：先做 BLC，再做 LSC、AWB 白点标定和 CCM 标定。
3. 亮度调优：以 [[wiki/modules/FH833X_AE|FH833X_AE]] 为主，联查 [[wiki/modules/FH833X_Gamma|FH833X_Gamma]]、[[wiki/modules/FH833X_DRC|FH833X_DRC]]。
4. 色彩调优：先 [[wiki/modules/FH833X_AWB|FH833X_AWB]]，再 [[wiki/modules/FH833X_CCM|FH833X_CCM]]，最后 [[wiki/modules/FH833X_CE_LC|FH833X_CE_LC]] / [[wiki/modules/FH833X_CNR|FH833X_CNR]] / [[wiki/modules/FH833X_PURPLE|FH833X_PURPLE]] / [[wiki/modules/FH833X_FC|FH833X_FC]] / [[wiki/modules/FH833X_HLR|FH833X_HLR]] 等辅助模块。
5. 对比度和通透性：通过 [[wiki/modules/FH833X_Gamma|FH833X_Gamma]]、DRC、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、[[wiki/modules/FH833X_YIE|FH833X_YIE]] 调整。
6. 清晰度和噪声：围绕 [[wiki/modules/FH833X_NR|FH833X_NR]]、[[wiki/modules/FH833X_APC|FH833X_APC]]、[[wiki/modules/FH833X_YNR|FH833X_YNR]]、[[wiki/modules/FH833X_CNR|FH833X_CNR]]、[[wiki/modules/FH833X_DPC|FH833X_DPC]] 做联动平衡。
7. WDR 模式：先接入与标定，再调 [[wiki/modules/FH833X_WDR|FH833X_WDR]]、AE、DRC、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、NR 和 APC。

## 通用风险在 FH833X 上的模块映射
- 通用风险总表见 [[wiki/workflows/跨平台_ISP_通用风险点检查表|跨平台 ISP 通用风险点检查表]]。FH833X 的风险类型并不特殊，主要差异在模块命名和控制链路。
- 黑电平 / 暗场基线：优先看 [[wiki/modules/FH833X_BLC|FH833X_BLC]]，黑电平不准会直接影响颜色、对比度和后续调试。
- 曝光与时序：优先看 [[wiki/modules/FH833X_AE|FH833X_AE]]，重点确认 intt、sensor gain、isp gain 生效时序，时序不一致会造成闪烁。
- WDR / HDR 合成：优先看 [[wiki/modules/FH833X_WDR|FH833X_WDR]]，短帧亮度矫正和曝光生效时机不一致会造成高亮闪烁或合成区异常。
- 降噪与拖影：优先看 [[wiki/modules/FH833X_NR|FH833X_NR]] 和 [[wiki/modules/FH833X_SHARP_NOISE|FH833X_SHARP_NOISE]]，NR3D 强度过大容易产生拖影。
- 锐化与噪声：优先看 [[wiki/modules/FH833X_APC|FH833X_APC]]，锐化会放大噪声，必须和 NR3D / NR2D / [[wiki/modules/FH833X_YNR|YNR]] / [[wiki/modules/FH833X_CNR|CNR]] 联调。
- 动态范围与色彩观感：优先看 [[wiki/modules/FH833X_DRC|FH833X_DRC]]、[[wiki/modules/FH833X_LCE|FH833X_LCE]]、[[wiki/modules/FH833X_Gamma|FH833X_Gamma]] 和 [[wiki/modules/FH833X_CCM|FH833X_CCM]]，调色前要确认亮度映射、饱和度抑制和色度补偿状态。
