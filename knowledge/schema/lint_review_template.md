# 知识库 Lint 审查模板

用于定期检查知识库结构、链接、来源和内容一致性。复制到 `logs/YYYY-MM.md` 中追加即可。

```markdown
## YYYY-MM-DD Lint 审查记录

### 本次范围
- `wiki/modules/`
- `wiki/issues/`
- `wiki/platforms/`
- `wiki/indexes/`

### 检查项
- [ ] 是否存在重复或同义页面
- [ ] 是否存在孤立页面，缺少索引入口或双向链接
- [ ] 页面属性是否缺失或与内容不一致
- [ ] 重要结论是否缺少 `raw/` 来源或案例来源
- [ ] 模块页是否缺少调试视角、参数联动和典型调试取舍
- [ ] 问题页是否缺少现象、可能原因、排查顺序、判断依据和常见误判
- [ ] 平台特有内容是否误写入通用模块页或通用问题页
- [ ] 案例经验是否被错误上升为无条件通用规律
- [ ] 索引页是否遗漏新页面或仍指向旧页面
- [ ] 双向链接是否缺失、失效或命名不统一
- [ ] 新资料是否修正或推翻旧结论

### 发现问题
- 

### 已处理
- 

### 待确认
- 
```
